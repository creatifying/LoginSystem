## Deployment

To deploy this project, Extract the .zip file onto your server, Edit the config.php file for setting up yout database configration

```bash
$server = "localhost"; 
$username = "root";
$password = "";
$dbname = "loginform";
 ```
Change thee details to yours and you're good to go !
